== Description ==

Simple Spam Blcoker stop spam comments and also can be used to stop bots to try to login into admin panel.

== Installation ==

1. Upload and unzip `spam-blocker.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Search in the left admin menu Spam Blocker to see its settings
